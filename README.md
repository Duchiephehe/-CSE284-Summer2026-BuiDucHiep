## Về Repository này

Repository này là nơi lưu trữ mã nguồn cho môn học, bao gồm hai phần chính:

1. **Bài tập các buổi:** Chứa các bài tập thực hành lập trình Java được giao theo từng buổi học (ví dụ: bảng cửu chương, vẽ mô hình hóa lớp , ... ).
2. **Bài tập lớn (BTL) - Cửa hàng Hoa Quả:** Một dự án nhỏ với chủ đề quản lý cửa hàng hoa quả. Hệ thống được thiết kế xoay quanh 3 đối tượng cốt lõi:
   - `Fruit` (Hoa quả): Quản lý thông tin về các mặt hàng sản phẩm.
   - `Customer` (Khách hàng): Quản lý thông tin của người mua hàng.
   - `Order` (Đơn hàng): Quản lý các giao dịch mua bán (liên kết khách hàng và hoa quả đã mua).
